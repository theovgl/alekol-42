- [Axios](https://github.com/axios/axios)
- [NodeJS](https://github.com/nodejs/node)

## Installation

Create a `.env` files with the following variables and your credentials:
```
DISCORD_TOKEN=
DISCORD_CLIENT_ID=
DISCORD_GUILD_ID=

SUPABASE_URL=
SUPABASE_ANON_KEY=

42_UID=
42_SECRET=
```
Install node packages and run the program:
```sh
npm i
npm run dev
```
