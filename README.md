- [NodeJS](https://github.com/nodejs/node)

## Installation

Rename the `.env.template` file and fill in the fields with your credentials:

```
DISCORD_TOKEN=
DISCORD_CLIENT_ID=
DISCORD_GUILD_ID=

SUPABASE_URL=
SUPABASE_ANON_KEY=

FT_USER_ID=
```

Install node packages and run the program:

```sh
npm i
npm start
```
